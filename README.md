# whatsappead0r
Generador de links whatsAppWeb múltiples

- Completar con los números (el +549 ya esta incluido) sólo el numero con código de área sin 15 (Ej. 1123456789). 
- Seleccionar el tipo de separador: Se pueden incluir números separados por comas (Ej. 1111111111,2222222222,3333333333) o por espacios o saltos de línea (Ej. 4444444444 5555555555 6666666666). (Sí, se pueden copiar y pegar directamente de un excel :D )
- Incluir un mensaje (opcional)
- La app generará un botón por cada número (requiere mínimo 2 números)
- Al usarlos abre una ventana de whatsApp Web en el navegador, teniendo instalado el programa de WP en el sistema operativo funciona mejor y mas rápido ya que evita tener que volver a cargar la sesion de WP Web en cada pestaña nueva
- Los botones que ya fueron clickeados cambian de color



* Para reutilizar el código con otro prefijo de país solamente deben modificar la constante urlUno
